System Recovery Tools
⚠️ Recovery tools make permanent changes to your OpenWebOS data.
Only use these if you are locked out or experiencing issues.
Account
Clear Saved Data
Wipes your name, password, wallpaper, and settings. You'll go through setup again.
Password
Reset your password without wiping other settings.
Files
Clear Uploaded Files
Deletes all files stored in OpenWebOS Files (IndexedDB). Settings are kept.
Diagnostics
View Saved Data
Inspect what's currently stored in localStorage.
← Close this tab and reopen OpenWebOS to apply changes
